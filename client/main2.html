Learning Bitcoin

Step 0: Set Up

Open the developer console with Option-Command-J

Drag the window to a comfortable size

Increase the text size with Command-Plus, or decrease with Command-Minus

When you're ready, type next() and press enter.

Step 1: Generate Mnemonic

Nice!

Now we're going to generate a 12-word mnemnoic

This will be the secret that controls your bitcoin wallet

Try the command:



words = bip39.generateMnemonic()



When you've done that, type next() and press enter.

Step 2: Generate Seed

Awesome!

The following 12 words are your secret mnemnonic:







If you can remember these, you can store bitcoin in your head!

Now let's convert this mnemonic into a seed. Try doing:



seed = bip39.mnemonicToSeedSync(words)



When ready, do next()

Step 3: Generate Address

Now lets create a wallet from your seed with:



wallet = bip32.fromSeed(seed)



Then create an address with:



pubkey = wallet.publicKey

network = bitcoin.networks.testnet

p2pkh = bitcoin.payments.p2pkh({pubkey, network})

address = p2pkh.address
